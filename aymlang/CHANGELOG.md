# Change Log

All notable changes to the "aymlang" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

- Pendiente de publicación.

## [0.0.4]

- Alinea la extensión con la variante canónica actual del lenguaje.
- Actualiza keywords de control a `ukaxa`, `maysatxa`, `ukhakamaxa`, `kuti`, `siqicha`, `khiti`, `kuna`, `yaqha`, `chiqa` y `k'ari`.
- Elimina aliases y keywords POO retiradas (`taqi`, `jikxata`, `uñt'aya`, `chura`) del autocompletado y resaltado.

## [0.0.3]

- Actualiza palabras clave y funciones integradas para la sintaxis actual.
- Mejora snippets con `ukaxa`, `maysatxa`, `ukhakamaxa`, `kuti` y `katu`.
