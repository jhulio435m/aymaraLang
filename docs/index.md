# AymaraLang

Documentación esencial para instalar, escribir y compilar programas en
AymaraLang sin navegar por material interno innecesario.

## Si es tu primera vez

1. [Instalación](install.md)
2. [Tutorial rápido](language.md)
3. [Manual de usuario](manual_usuario.md)

## Qué encontrarás aquí

- Cómo instalar `aymc` y `aym` en Windows y Linux.
- Cómo escribir tus primeros programas `.aym`.
- Cómo compilar, validar y ejecutar proyectos.
- Una referencia breve del lenguaje y una gramática formal.

## Ruta recomendada

- Si solo quieres usar el lenguaje: [Instalación](install.md) -> [Tutorial rápido](language.md) -> [Manual de usuario](manual_usuario.md)
- Si necesitas opciones de compilación: [CLI de `aymc`](compiler.md)
- Si necesitas detalle sintáctico: [Referencia rápida](aymaraLang.md) y [Gramática formal](grammar.md)
- Si vas a compilar el compilador desde el repo: [Compilación desde fuente](build.md)
