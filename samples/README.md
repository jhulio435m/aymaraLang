# Samples

- `samples/fundamentos`: sintaxis base, funciones y ciclos.
- `samples/colecciones`: listas, mapas y operaciones de colecciones.
- `samples/control_flujo`: `match`, enums y variantes de flujo.
- `samples/poo`: ejemplos de orientacion a objetos.
- `samples/modulos`: imports y validaciones de modulos.
- `samples/apps`: programas completos.
- `samples/games`: juegos (`console` y `gui`).
- `samples/bench`: benchmark del compilador.
