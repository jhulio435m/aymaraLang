# Samples

- `samples/fundamentos`: sintaxis base, funciones y ciclos.
- `samples/colecciones`: listas, mapas y operaciones de colecciones.
- `samples/control_flujo`: `khiti`, `siqicha` y variantes de flujo.
- `samples/poo`: ejemplos de orientacion a objetos.
- `samples/modulos`: imports y validaciones de modulos.
- `samples/apps`: programas completos.
- `samples/games`: juegos (`console` y `gui`).
- `samples/bench`: benchmark del compilador.

Nuevos ejemplos recomendados:

- `samples/control_flujo/excepciones_tukuyawi.aym`: flujo `yant'aña/katjaña/tukuyawi`.
- `samples/colecciones/arrays_ciclo_vida.aym`: ciclo de vida de `array` y `array_length`.
- `samples/fundamentos/matematica_builtins_check.aym`: validacion ejecutable de built-ins matematicos.
